# Pixelssuite Chat Translator - Automated UI Testing (QA)

This project contains an automated test suite for validating the transliteration accuracy of the Chat Sinhala Translator (Singlish to Sinhala). The tests are written in Python using Playwright for end-to-end (E2E) UI interaction and `openpyxl` for reading and writing test cases from an Excel spreadsheet.

## Requirements

- Python 3.10+
- Playwright
- openpyxl

## Installation Instructions

1. **Clone the repository** (or extract the zip file):
   ```bash
   git clone <your-repository-url>
   cd IT23225060playwright
   ```

2. **Set up a Virtual Environment**:
   ```bash
   python -m venv .venv
   ```

3. **Activate the Virtual Environment**:
   - On Windows:
     ```bash
     .\.venv\Scripts\activate
     ```
   - On macOS/Linux:
     ```bash
     source .venv/bin/activate
     ```

4. **Install Dependencies**:
   ```bash
   pip install playwright openpyxl
   ```

5. **Install Playwright Browsers**:
   ```bash
   playwright install chromium
   ```

## Running the Tests

To run the automated test suite against the target web application:

1. Ensure your virtual environment is activated.
2. Execute the test automation script:
   ```bash
   python IT23225060_test_automation.py --excel "IT23225060_Assignment 1 - Test cases.xlsx" --output "IT23225060_Assignment 1 - Test cases.xlsx"
   ```

### Optional Arguments
- `--headless`: Run the browser in headless mode (no UI).
- `--slow-mo-ms <milliseconds>`: Slow down execution to watch the interactions.
- `--keep-open`: Keep the browser open after tests finish.

Example:
```bash
python IT23225060_test_automation.py --excel "IT23225060_Assignment 1 - Test cases.xlsx" --slow-mo-ms 500
```

## Results
The script will iterate through the provided Singlish test cases, input them into the web UI, extract the transliterated Sinhala output, and automatically mark the test status in the Excel file.
